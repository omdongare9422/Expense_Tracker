### Prerequisites
- Node.js installed.

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/omdongare9422/Expense_Tracker.git
    cd Expense_Tracker
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run locally**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
